# Zoon 商业化启动清单：个人模式 + Team 协作模式

这份清单用于启动第一期商业化。当前只做两种模式：

1. 个人模式：低价个人入口，让用户快速完成第一次局部 Agent 改稿。
2. Team 协作模式：季度 Team Pilot，验证文本团队是否愿意为多人 + 多 Agent 改稿流程付费。

本期不卖：

- Team Pilot Plus。
- 成熟 Team Pro。
- Team Scale。
- 企业治理年费。
- 私有部署。

## 1. 需要准备什么

### A. Offer 和定价

个人模式：

- ¥19.9 / 季度：轻量 quick actions。
- ¥29 / 月或 ¥99 / 季度：更多 quick actions + 少量深度改稿。
- 卖点：开箱即用的 Zoon Agent，解决“我不知道怎么让 Agent 帮我改这一段”。

Team 协作模式：

- ¥1,999 / 季度。
- 卖点：帮文本团队跑通“多人 + 多 Agent + Markdown 改稿 + 来源可见”的真实协作流程。
- 边界：文本为主，不承诺图片、完整团队后台、SSO、私有部署、正式审计、SLA。

需要你决策：

- 个人模式最终价格选 ¥19.9 / 季度，还是 ¥29 / 月 / ¥99 季度。
- Team Pilot 是否坚持 ¥1,999，不做免费试点。
- 是否接受对方用脱敏文本开始试点。

我可以执行：

- 写定价说明。
- 写销售话术。
- 写反对意见处理。
- 把页面 CTA 和销售材料统一到这两个模式。

- 1 个“局部追问”入口。
- 1 个“保留 / 删除 AI 修改”的清晰流程。
- 1 份个人模式 Demo 文档。
- 1 段个人模式销售文案。
- 1 套免费额度 / 付费额度规则。

需要你配合：

- 决定是否接入 DeepSeek 作为内置 Zoon Agent。
- 提供 DeepSeek API key 或确认先不接模型，只用外部 Agent。
- 决定个人模式是否马上收费，还是先做灰度激活。
- 提供 1-2 篇真实但可脱敏的个人改稿素材。

- 至少 3 个用户理解“Zoon 是让 Agent 贴着原文改，不是普通编辑器”。

      ├─ 价格：¥1,999 / 季度

### 首页是否要改

需要改，但不要一次改成完整商业化官网。首页第一版只承担两个任务：

1. 让个人用户立刻理解：Zoon 不是普通 Markdown 编辑器，而是“选中一段，让 Agent 直接改”。
2. 让团队用户知道：如果你们有大量文本稿件 / PRD / 方案，可以申请季度 Team Pilot。

当前首页更偏“和 Agent 一起写成文档 / 产品工作站”。这个方向不算错，但对当前商业化太宽。建议第一屏改成更窄：

标题建议：

> 让 Agent 直接改你的 Markdown 稿件。

副标题建议：

> 不用在 AI 和飞书之间来回复制。选中一段，修语法、改表达、缩短或让 Agent 评论；团队可以在同一份文档里审校和保留修改。

两个 CTA：

- 免费创建文档，试一次局部改稿。
- 申请季度 Team Pilot。

首页需要新增或调整的模块：

- 个人模式：公众号作者、内容编辑、产品经理写方案各一个场景卡。
- Team 协作模式：多人 + 多 Agent 改稿流程说明。
- 明确边界：文本为主，不处理图片排版，不是飞书 / Google Docs 替代品。
- 简单价格入口：个人模式价格待定；Team Pilot ¥1,999 / 季度。

首页暂时不要放：

- Team Pro / Team Scale。
- 企业治理、SSO、私有部署。
- 太重的 provenance / audit 叙事。
- 复杂套餐表。

### 除首页外还要准备什么

必须准备：

- 新用户 onboarding：第一次打开文档时，引导完成一次局部 Agent 改稿。
- Quick actions：修复语法、改善表达、缩短必须稳定可见。
- 没有外部 Agent 的 fallback：如果本期接 DeepSeek，就用内置 Zoon Agent；如果不接，就把“复制给外部 Agent”的流程做得更傻瓜。
- Team Pilot 申请入口：可以先是表单、邮箱、微信或一个简单按钮，不需要完整 CRM。
- Demo 文档：公众号长文、内容编辑稿、产品方案各一份。
- 轻量指标：记录创建文档、first agent collaboration、Agent 写入 / 评论 / 建议、保留 / 删除。

暂时不要准备：

- 完整 billing 系统。
- 完整团队后台。
- 企业审计报告。
- 图片工作流。
- 私有部署方案。

### H. FAQ

#### Zoon 是飞书 / Google Docs 替代品吗？

```text
不是。Zoon 当前更适合作为 AI 改稿中间工作台。你可以在 Zoon 里让 Agent 贴着原文改，最后再迁回飞书、公众号编辑器或其他交付工具。
```

#### Zoon 自己提供 AI 模型吗？

```text
Zoon 的核心是让 Agent 进入文档协作。个人模式可以接入内置 Zoon Agent；高级用户也可以继续使用 Claude Code、Codex、Cursor、ChatGPT 或其他能发 HTTP 请求的 Agent。
```

#### 为什么用 Markdown？

```text
Markdown 对人类够简单，对 Agent 也足够清晰。它让 AI 更容易准确读、改、评论你正在写的那一段。
```

#### 支持图片和排版吗？

```text
当前商业化试点只承诺文本密集工作流。图片、富媒体排版和企业级文档管理不在本期范围内。
```

#### Team Pilot 适合谁？

```text
适合每周有 5 份以上 AI 参与文本稿件、PRD、方案或报告的团队。如果你们最大痛点是图片、海报或复杂排版，这期不适合。
```
