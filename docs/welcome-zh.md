# 欢迎使用 Zoon

Zoon 是一个人机协作文档编辑器——你和 AI 一起写，每一段文字都清楚地知道是谁写的。

---

## Zoon 是怎么工作的

**溯源（Provenance）** — 左侧彩色边栏追踪内容来源。绿色 = 你写的，紫色 = AI 写的。删改、插入，每个字都有作者。

**批注（Comments）** — Agent 不会直接修改你的文档。它会在你写的文字上添加批注，说明想改什么、为什么改，然后等你拍板。

**👍 确认协议** — 这是 Zoon 的核心工作方式：

```
你 → 加批注或直接告诉 Agent 你想做什么
Agent → 在批注里回复方案，附上建议修改的内容
你 → 回复 👍，Agent 立即执行修改
你 → 回复 👎 或追问，Agent 重新提方案
```

你始终掌握最终决定权。Agent 绝不会绕过你直接修改文档。

---

## 邀请 Agent 协作

点击右上角「邀请 Agent」，复制提示词，粘贴给你的 AI 工具（Claude Code、Cursor、ChatGPT 等）。

Agent 会自动读取文档内容，然后等你告诉它接下来要做什么。

**支持所有主流 AI 工具：** Claude Code、Codex、Cursor、ChatGPT，以及任何可以发 HTTP 请求的 Agent。

---

## 可以做什么

Agent 连接后，试着让它：

- **整理笔记** — 把你记录的会议内容、想法碎片整理成结构清晰的文档
- **扩展草稿** — 给它一句话或一个提纲，让它补全完整段落
- **审阅文章** — 检查逻辑漏洞、找出可以更精准的表达
- **头脑风暴** — 就一个问题列出所有可能的角度和方向
- **提炼要点** — 把长文章压缩成 3-5 个核心观点
- **对话推进** — 直接在批注里来回讨论，不用切换到其他工具

## 协作示例

**场景：改进这段话**

> 你在文档里写了一段说明，感觉有点啰嗦

1. 选中这段文字，在左侧边栏加一条批注：「帮我精简这段」
2. Agent 在批注 thread 里回复：「建议改成：XXX，回 👍 我就改」
3. 你看一眼，回 👍
4. Agent 立即完成修改，回复「已改 ✓」，批注自动关闭

整个过程不离开文档界面。
