# 欢迎使用 Zoon

Zoon 是给 Agent 输出用的 Markdown 编辑器。你可以把 AI 写好的方案、PRD、文章放进来，像普通文档一样继续编辑。需要 AI 帮忙时，选中一段，让它按你的要求重写、补充、缩短或改成你的语气。

---

## Zoon 是怎么工作的

**放入 AI 初稿** — 把 Codex、Claude Code、Cursor 或 ChatGPT 写好的 Markdown 粘进来，先变成一份能继续编辑的文档。

**选中一段再让 AI 改** — 不用重新解释整篇上下文。你选中哪里，就让 AI 只围绕那里重写、补充、缩短或调整语气。

**人工继续审稿** — AI 给出修改后，你可以继续改、删、保留，直到这份 Markdown 真正能交付。

**修改过程清楚** — 不怕 AI 把原稿改乱；你可以看到哪些地方被处理过，也可以选择用评论或建议先讨论。

---

## 邀请 Agent 协作

点击右上角「邀请 Agent」，复制提示词，粘贴给你的 AI 工具（Claude Code、Cursor、ChatGPT 等）。

Agent 会加入文档并报一句"准备好了"，然后等你告诉它要改哪一段、怎么改。

**支持所有主流 AI 工具：** Claude Code、Codex、Cursor、ChatGPT，以及任何可以发 HTTP 请求的 Agent。

---

## 可以做什么

Agent 连接后，试着让它：

- **继续编辑 AI 初稿** — 把一份方案、PRD 或文章改到能真正使用
- **局部定制修改** — 选中一段，让 AI 按你的要求重写、补充、缩短或换语气
- **人工审稿** — 你继续改正文，AI 只帮你处理需要帮忙的部分
- **补充段落** — 让 AI 在指定位置补一段背景、案例、风险或下一步
- **改成交付版本** — 把看起来完整但还不够准的 AI 输出，改成可以发给同事、客户或读者的版本

---

## 协作示例

**场景 A：把 AI 初稿改成你的版本**

> Codex 先写了一份产品方案，但有一段太工程化

1. 把方案放进 Zoon。
2. 选中那一段，告诉 Agent「改成产品经理能读懂的用户价值表达」。
3. Agent 只改这段；你再人工审一遍，保留合适的版本。

**场景 B：让 AI 补一段缺失内容**

> PRD 里还缺一段风险说明

1. 把光标放到要补充的位置。
2. 告诉 Agent「补一段上线风险和兜底方案，语气简洁」。
3. Agent 写进去后，你继续删改到符合你的判断。

你可以让 Agent 直接处理正文；如果你想先看提案，也可以要求它用建议或评论。
