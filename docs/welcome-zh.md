# 欢迎使用 Zoon

Zoon 是一个人机协作文档编辑器——你和 AI 一起写，每一段文字都清楚地知道是谁写的。

---

## Zoon 是怎么工作的

**溯源（Provenance）** — 左侧彩色边栏追踪内容来源。绿色 = 你写的，紫色 = AI 写的。删改、插入，每个字都有作者。

**Agent 直接写进文档** — Agent 不需要模拟浏览器，也不需要先走审批。它通过 HTTP 读文档、写文档、发 presence、留评论。因为作者来源可见，你能看到是谁改了什么。

**评论和建议是主动选择** — 默认是直写；如果你希望先讨论、先审阅，就明确让 Agent 用 comment 或 suggestion。Zoon 不强制把修改人类原文变成审批流。

---

## 邀请 Agent 协作

点击右上角「邀请 Agent」，复制提示词，粘贴给你的 AI 工具（Claude Code、Cursor、ChatGPT 等）。

Agent 会加入文档并报一句"准备好了"，然后等你告诉它要做什么方向。

**支持所有主流 AI 工具：** Claude Code、Codex、Cursor、ChatGPT，以及任何可以发 HTTP 请求的 Agent。

---

## 可以做什么

Agent 连接后，试着让它：

- **整理笔记** — 把你记录的会议内容、想法碎片整理成结构清晰的文档
- **扩展草稿** — 给它一句话或一个提纲，让它补全完整段落
- **审阅文章** — 检查逻辑漏洞、找出可以更精准的表达
- **头脑风暴** — 就一个问题列出所有可能的角度和方向
- **提炼要点** — 把长文章压缩成 3-5 个核心观点
- **对话推进** — 直接在批注里来回讨论，不用切换到其他工具

## 协作示例

**场景 A：让 AI 扩写一段**

> 你写了一个提纲，想让 AI 补成完整段落

1. 在提纲下方空一行，告诉 agent「顺着这个大纲写一段，放在这里」
2. Agent 直接把新段落写进去——紫色，一眼能看出是它写的
3. 你扫一遍，觉得第二句啰嗦？点那句紫字，直接改或删就行

**场景 B：让 AI 改你原文里一个词**

> 你自己写的一句话，想换个说法

1. 直接告诉 Agent：「把这句话换个更具体的说法」
2. Agent 直接改正文，并带上 `ai:<agent-name>` 作者身份
3. 如果你想先审阅，就说：「用 suggestion 提案，不要直接改」

直写是默认；评论和建议是你们主动选择的协作姿势。
